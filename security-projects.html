Cyber Security Projects

Project Title

Team academic project in Cyber Security with Barak Sofir, supervised by Amichai Shulman

In this project we implemented a PoC of a super resilient C&C two-way communication based on public infrastructure in a cost-effective way. The C&C is based on comments on Reddit content and is extremely difficult to detect and very hard to dismantle.

If you'd like to explore the project in more detail, please don't hesitate to peruse the project report.

The project was nominated for an Excellent Project Award in the Computer Science Faculty at the Technion. If you are interested, please check the project poster and the letter of recommendation from Amichai Shulman in support of the project's nomination.

technologies: Python, Selenium, Ecdsa
